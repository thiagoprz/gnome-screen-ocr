## Description
A GNOME Shell extension that allows users to select any area on the screen
and copy recognized text using Tesseract OCR.

Inspired by Google Lens on Android.

## Dependencies
- gnome-screenshot (Wayland safe)
- tesseract-ocr (+ language packs)
- imagemagick
